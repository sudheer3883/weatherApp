# 🌦️ Weather App

A responsive Weather App built using HTML, CSS, and JavaScript.

## 🚀 Features

- Search weather by city name
- Real-time weather information
- Weather condition icons
- Responsive design
- Easy to use interface

## 🛠️ Technologies Used

- JavaScript
- Weather API
